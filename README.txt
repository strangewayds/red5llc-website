HOW TO OPEN THE WEBSITE
-----------------------------------------------------------------
Easiest way:
   Double-click  "index.html"  in this folder.
   It opens in your web browser. Every page and button works.

Nicer way (a real web address, http://localhost:5500):
   1. Double-click  "start-website.bat"
   2. A small black window opens (leave it open).
   3. In your browser go to:  http://localhost:5500
   To stop it, just close that black window.

HOW TO SWAP IN BETTER PHOTOS OF ROSIE
-----------------------------------------------------------------
All photos live in the "assets" folder. To replace one, just put
your new picture in that folder using the SAME file name:

   rosie-hero.png .... big photo on the Home page (the hero)
   rosie-1.png ....... Meet Rosie photo #1 (left)
   rosie-2.png ....... Meet Rosie photo #2 (middle)
   rosie-3.png ....... Meet Rosie photo #3 (right)
   logo.png .......... the Red 5 logo (top-left of every page)

Tip: keep the same file name and it will appear automatically.
(.jpg works too — if you use a .jpg, tell me and I'll point the
 page at it.)
